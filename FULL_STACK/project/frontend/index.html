Optional static frontend

The main UI is server-rendered at /events on the Express app (same folder as backend/). For a JSON API example served by the backend, open http://localhost:3000/events-api-demo.html after you log in (uses GET /api/events with cookies).

Client boilerplate:

const res = await fetch("http://localhost:3000/api/events", {
  credentials: "include", // send login cookie; same-site only unless you add CORS
});
const data = await res.json();
// data.events → array of events
